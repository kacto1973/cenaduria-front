📝 README: Sistema de Gestión "Los Álamos"

Descripción:
Aplicación web móvil-first diseñada para la administración integral de ventas y flujo de comensales en la Cenaduría Los Álamos. El sistema centraliza la toma de pedidos, el control de mesas y la seguridad financiera del establecimiento.

Stack Tecnológico:

    Frontend: React.js (Vite) con Tailwind CSS (Diseño formal y responsivo).

    Backend: Node.js + Express.

    Base de Datos: MongoDB Atlas (NoSQL).

    Seguridad: Validación de Red Wi-Fi y Autenticación por PIN.

Objetivo:
Optimizar la atención al cliente y eliminar el "robo hormiga" mediante la trazabilidad total de cada orden y la restricción geográfica de uso.
